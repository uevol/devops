# devops

[devops](http://180.173.34.78:8007/) a plateform for cmdb, saltstack, zabbix and systemlog

License: MIT

#### 已完成模块

>一、 账户管理

>> 1. 用户管理

>> 2. 用户组管理

>二、权限管理

>> 1. 权限列表

>> 2. 角色管理

>三、资产管理

>> 1. 模型管理 -- 可以自定义模型属性

>> 2. 实例列表

>四、作业平台（基于saltstack开发）

>> 1. 命令执行 -- 批量执行命令

>> 2. 文件分发 -- 批量向远端分发文件

>> 3. 脚本管理 -- 批量执行脚本

>> 4. 模块部署 -- 复杂任务通过saltstack模块部署

>> 5. 定时任务 -- 定时执行命令或脚本（类似cron）

>> 6. 操作记录 -- 以上操作的均会记录，方便查询
